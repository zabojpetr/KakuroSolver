\section*{Předdefinované příklady}
Program obsahuje několik příkladů kakura.
Pokud je chete spustit stačí zadat \haskellline/kakuro <kakuro>/, kde místo $<$kakuro$>$ zadáte jednu z možností:\\
Hotová kakura
\begin{enumerate}
\item dobre
\item spatne
\item vyresene
\end{enumerate}
Navyplněná kakura
\begin{enumerate}
\item easy
\item easy2
\item medium
\item nevyresene
\item extreme
\item extreme2
\item extreme3
\end{enumerate}
